This page looks very empty. But that's good, because we will learn the basics of the most important web technologies in the next pages and this page will grow bit by bit. We start with the basics HTML, then add styles (CSS) and logic (JavaScript). With this base we start with the visualisation of data with Scalable Vector Graphics (SVGs) and D3 then.

In this course we learn visualizing data with Data-Driven Documents (D3). It is a very powerful library, maybe the best. Here are some examples what you can build with it:

  • Simple Barcharts
  • Scatterplots
  • Interactive Data
  • Maps
  • Force directed Graphs
  • And a lot more fancy things!
